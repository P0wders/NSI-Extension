# NSI-Extension
Une extension firefox designer pour faciliter les réponses de cours pour le site raisintine.

> [!IMPORTANT]
> Cette extension n'a été tester uniquement sur firefox, et donc peut être sujet a de nombreux bug sur d'autres navigateurs...

Cette extension est open-source et ne nécésite aucune lisence, elle est totalement libre de droit, elle peut donc être modifier de tout les façons
